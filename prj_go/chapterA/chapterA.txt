由于new只是一个预定义的函数，它并不是一个关键字，
因此我们可以将new名字重新定义为别的类型。 例如下面的例子： 
func delta(old, new int) int { return new - old }
由于new被定义为int类型的变量名，因此在delta函数内部是无法使用内置的new函数的。

如果两个类型都是空的，也就是说类型的大小是0，例如struct{}和 [0]int, 有可能有相同的地址
